**Project is under active development, stay tuned for improved features, documentation, and releases!**

Meshwell uses [OpenCASCADE](https://dev.opencascade.org/) for CAD and wraps [GMSH](https://gmsh.info/) and [MMG](https://github.com/MmgTools/MMG) for meshing and remeshing. It streamlines the definition of geometries parametrized by [Shapely](https://shapely.readthedocs.io/en/stable/) polygons (and arbitrary OCC shapes) into high-quality meshes.
